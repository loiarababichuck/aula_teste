# Primeira Atividade com Git

## 📌 Descrição

Este repositório foi criado como parte da minha primeira atividade prática utilizando **Git**. O objetivo foi aprender a configurar o Git no computador e utilizar seus comandos básicos para controle de versão.

## 🎯 Objetivos da atividade

* Instalar o Git no computador
* Configurar as informações do usuário
* Criar um repositório
* Realizar o primeiro commit
* Enviar arquivos para um repositório remoto

## ⚙️ Configurações iniciais do Git

Para começar a utilizar o Git, foram realizadas as seguintes configurações no terminal:

```bash
git config --global user.name "Seu Nome"
git config --global user.email "seuemail@email.com"
```

Para verificar as configurações realizadas:

```bash
git config --list
```

## 📂 Criação do repositório

Inicialização do repositório Git:

```bash
git init
```

## 📥 Adicionando arquivos

Adicionar arquivos para o controle de versão:

```bash
git add .
```

## 💾 Realizando o primeiro commit

```bash
git commit -m "Primeiro commit"
```

## 🚀 Enviando para o GitHub

```bash
git remote add origin URL_DO_REPOSITORIO
git branch -M main
git push -u origin main
```

## 🛠️ Tecnologias utilizadas

* Git
* GitHub

## 📚 Aprendizado

Com esta atividade foi possível aprender os conceitos básicos de **controle de versão**, além de compreender como versionar arquivos e enviar projetos para um repositório remoto.
